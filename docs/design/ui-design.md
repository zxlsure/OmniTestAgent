## 1. 前端架构概览

### 1.1 技术栈

| 技术 | 版本 | 用途 |
|------|------|------|
| Vue 3 | 3.5.x | 前端框架 (Composition API + `<script setup>`) |
| TypeScript | 5.8.x | 类型安全 |
| Arco Design Vue | 2.x | UI 组件库 |
| Pinia | 3.x | 状态管理 |
| Vue Router 4 | 4.x | 路由管理 |
| Vite | 6.x | 构建工具 + 开发服务器 (渲染进程) |
| Tailwind CSS 4 | 4.x | 原子化 CSS |
| ECharts | - | 数据可视化 |
| markdown-it | - | Markdown 渲染 |

### 1.2 项目结构

```
src/
├── main.ts                     # 应用入口
├── App.vue                     # 根组件
├── router/
│   └── index.ts                # 路由配置 + 全局守卫

│   ├── useProjectStore.ts

│   ├── useTestFlowStore.ts

│   ├── knowledge/
```

## 2. 启动流程

```
index.html
  → main.ts
    → createApp(App)
    → app.use(Pinia)
    → app.use(Router)
    → app.use(ArcoVue) + app.use(ArcoVueIcon)
    → app.mount('#app')
  → App.vue
    → onMounted: useAppStore().initialize()
      → 检查 LLM 配置状态
      → 加载项目列表
      → 恢复上次选中的项目
    → <router-view />
```

## 6. IPC 调用封装 (useIpc composable)

```typescript
// composables/useIpc.ts
export function useIpc() {
  // 通用 IPC 调用 (invoke 模式)
  async function invoke<T>(channel: string, ...args: unknown[]): Promise<T> {
    return window.electronAPI[channel](...args)
  }

  // 事件监听 (on 模式)
  function on(channel: string, callback: (...args: any[]) => void): () => void {
    window.electronAPI.on(channel, callback)
    // 返回取消监听函数
    return () => window.electronAPI.off(channel, callback)
  }

  return { invoke, on }
}
```

## 9. Chat 交互设计

### 9.1 斜杠命令弹窗

输入 `/` 触发斜杠命令弹窗，显示 4 个命令（/analyze、/design、/cases、/script），支持模糊过滤和键盘导航。

### 9.2 @文件引用输入

消息输入框中输入 `@filename` 触发文件引用，Agent 执行时自动解析并注入文件内容到用户消息前。

### 9.3 审批面板

当 Agent 调用需审批的工具时，Chat 界面弹出审批面板，显示工具名称、参数，提供「批准」「拒绝」按钮。超时 120s 自动拒绝。

### 9.4 Agent 阶段指示器

Agent 执行过程中，消息区域显示阶段指示器：
- thinking：推理中（显示思考内容折叠区）
- tool_call：工具调用中（显示工具名和参数）
- result_organize：整理结果中
- done：完成
